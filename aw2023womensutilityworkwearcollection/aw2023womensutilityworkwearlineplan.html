This line plan was created to outline each one of the garments included in this workwear collection. Essential information such as estimated price, available colorways, size range and fabric choice are included under each piece of clothing to aid the future production process. I referenced the boards I created in the concept phase to finalize a color palette and thoughtfully chose which fabric was best suited to each piece. Illustrator was once again used to create this line plan.
