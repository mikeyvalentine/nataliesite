  The images below are some visual merchandising ideas I created for the brand Everlane. The main idea of this project was that Everlane was preparing to open a store in Portland, Oregon. Window displays, wall displays and an interior store display were a major part of the project followed by some different marketing tactics. I created an exterior view of a storefront under construction, an opening announcement on their website and an instagram post to get customers excited for the new store opening.

  The majority of this project was developed on Procreate.
